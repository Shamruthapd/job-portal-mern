# AI-Powered Job Portal (MERN)

A full-stack MERN Job Portal connecting job seekers and recruiters.

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## Features

### Job Seekers

* Register/Login
* Profile Management
* Resume Upload
* Browse Jobs
* Apply for Jobs
* Track Applications

### Recruiters

* Recruiter Login
* Post Jobs
* Manage Job Listings
* View Applicants

## Future AI Features

* AI Job Recommendation
* Resume Analyzer
* Career Guidance Chatbot
* Cover Letter Generator
